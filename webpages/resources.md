## SARGON Distribution
---

The SARGON a modular network ontology which defined as a set of [Turtle][3] and [OWL][4] files.
Turtle is a compact textual format that is understood by most Semantic Web tools.


- **Latest Version (1.1)**:
   - [SARGON.ttl][1]
   - [SARGON.OWL][2]
